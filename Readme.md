## DESIGN NOTES FOR CLAUDE CODE  
  
### Use design reference for layouts and typesetting.  
**Aesthetic reference :** Polestar.com  
**Tone:** Cinematic, design-led, confident. Not corporate.   
**Suggested tech stack:** HTML/CSS/JS with GSAP for scroll-triggered animations, or React + Framer Motion   
**Animation approach:**  
- Scroll-triggered section reveals  
- Staggered text and card entrances  
- The "shift table" strikethroughs should animate — old text strikes through, new text fades in  
- The mantra "Create to Connect™" should have a hero moment — large, centered, potentially with a subtle motion treatment  
- Callout blocks can slide in from the left (matching their left-border accent)  
- Pills/tags can stagger in with a pop  
  
**Color palette (from current design):**  
- Red accent: #d32127  
- Dark grey: #333333  
- light grey: #cccccc  
- Background: inter mix black (studio/cinematic feel) or clean off-white #e7e7e7  
- Text: high contrast with key phrases highlighted red  
  
**Navigation: See top bar navigation design reference for style. Navigation remains at top on scroll down.**  
**Section structure:** Single-page scroll with optional anchored nav   
**Typography:** Strong weight contrast between labels (small caps, muted) and headings (large, medium weight)
